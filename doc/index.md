# Fork-Nft 文档

**Fork-Nft** 是 FLVX 的通用转发管理分支，支持多引擎转发能力与增量运维流程。

> 📞 **联系我们**: [Telegram群组](https://t.me/flvxpanel)

## 核心特性

- **多协议支持**: 支持 TCP 和 UDP 转发。
- **转发模式**: 支持端口转发与隧道转发。
- **多引擎能力**: 支持转发级引擎选择（`gost` / `nftables` / `realm`）。
- **增量同步**: 支持从 PO0 状态文件同步现有转发到面板，并可锁定 `nftables`。
- **管理能力**: 支持配额、限速、批量下发与分组管理。
